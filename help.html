Accessibility Help

Use the accessibility options provided by this Website to control the screen display. These options allow increasing the text spacing, changing the text size and colour scheme for clear visibility and better readability.

Text Size Icons

Following different options are provided in the form of icons which are available on the top of each page:

  • or press ctrl + key on keyboard to increase the text size
  • or press ctrl - key on keyboard to decrease the text size

Photo Gallery (Keyboard only)

Tab around the photos page and focus will be visible at one of the thumbnail images.

Press the enter key while on a link (focus)

The enter key will trigger the action just as a mouse click, interact with the controls by clicking on right arrow key to view next images, left arrow key to view previous images and escape key to close the image or return to the photos page.
